对象使用

Object.is

Object.is(val,val1) // 比较是否一致 Object.is(NaN,NaN) // true

Object.assign

Object.assign(target,obj1,obj2) // 合并对象 浅拷贝 只拷贝自身属性，不拷贝继承属性和不可枚举的属性
// 要保持克隆继承的值的话

Object.getPrototypeOf() 方法返回指定对象的原型（内部[[Prototype]]属性的值
Object.create()方法创建一个新对象，使用现有的对象来提供新创建的对象的proto。

Object.getOwnPropertyDescriptors()

ES5 的Object.getOwnPropertyDescriptor()方法会返回某个对象属性的描述对象（descriptor）。ES2017 引入了Object.getOwnPropertyDescriptors()方法，返回指定对象所有自身属性（非继承属性）的描述对象。

Object.getOwnPropertyDescriptors()方法配合Object.defineProperties()方法，就可以实现正确拷贝
Object.getOwnPropertyDescriptors()方法配合Object.create()可以克隆 浅拷贝
Object.getOwnPropertyDescriptors()方法可以实现一个对象继承另一个对象。以前，继承另一个对象，常常写成下面这样。

proto属性，Object.setPrototypeOf()，Object.getPrototypeOf() ,Object.create()

Object.setPrototyperof(target,,prot)
Object.getPrototypeOf(obj)

Object.keys() Object.values Object.entries() // 遍历对象
